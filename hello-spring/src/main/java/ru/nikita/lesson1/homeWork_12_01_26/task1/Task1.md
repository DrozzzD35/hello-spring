
Создай Spring Boot проект со следующими endpoints:

1. `GET /` — возвращает "Добро пожаловать в мой первый Spring Boot API!"
2. `GET /info` — возвращает информацию о приложении (название, версию, автора)
3. `GET /greet/{name}` — возвращает персонализированное приветствие

**Требования:**
- Проект создан через Spring Initializr
- Используется Spring Boot 3.x и Java 17+
- Добавлен только `spring-boot-starter-web`

**Критерии:**
- [ ] Все три endpoint работают
- [ ] Приложение запускается без ошибок
- [ ] Код чистый и понятный
