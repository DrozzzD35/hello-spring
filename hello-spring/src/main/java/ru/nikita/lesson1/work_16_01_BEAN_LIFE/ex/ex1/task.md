### Основное задание

Создай сервис кэширования с жизненным циклом:

1. **`CacheService`** — singleton сервис
2. В `@PostConstruct` — загрузи данные из файла `cache.json` (или создай пустой кэш)
3. В `@PreDestroy` — сохрани кэш в файл
4. Методы: `put(key, value)`, `get(key)`, `size()`
5. **`CacheController`** — endpoints для работы с кэшем

**Требования:**
- Используй `@PostConstruct` и `@PreDestroy`
- Логируй все операции жизненного цикла
- Кэш должен переживать перезапуск (данные в файле)

**Критерии:**
- [ ] Данные сохраняются при остановке
- [ ] Данные загружаются при старте
- [ ] В логах видно lifecycle events
