Основное задание

Создай приложение с полноценной конфигурацией:

1. Создай `application.yml` со следующими настройками:
    - Настройки сервера (порт, context-path)
    - Настройки приложения (имя, версия, описание)
    - Настройки API (url, timeout, api-key из переменной окружения)

2. Создай `ApiProperties` с `@ConfigurationProperties`
    - Добавь валидацию (@NotBlank, @Min/@Max)

3. Создай endpoint `/config` который возвращает все настройки (кроме секретов!)

4. Создай два профиля:
    - `dev` — локальные настройки, debug логирование
    - `prod` — production настройки, warn логирование

**Критерии:**
- [ ] YAML корректно структурирован
- [ ] @ConfigurationProperties работает
- [ ] Валидация срабатывает на неверные значения
- [ ] Профили переключаются корректно

### ⭐ Задание со звёздочкой

Добавь горячую перезагрузку конфигурации:
1. Добавь `spring-boot-starter-actuator`
2. Включи endpoint `/actuator/refresh`
3. Добавь `@RefreshScope` на бин
4. Измени значение в properties и вызови refresh
